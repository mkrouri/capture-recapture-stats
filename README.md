# Capture-Recapture Stats

DM - Master ISN, Universite de Lille (2025)
Melinda Krouri & Charlie Oudinot

## Objectif
Estimer la taille N d'une population via les modeles de capture-recapture,
approche frequentiste et bayesienne.

## Methodes

### Approche frequentiste
- Derivation de l'estimateur de Lincoln-Petersen par maximum de vraisemblance
- Analyse du biais et de la variance par simulation

### Approche bayesienne
- Implementation d'un algorithme MCMC (Metropolis-Hastings)
- Estimation de la distribution a posteriori de N et p
- Intervalles de credibilite a 95%

## Donnees
Application sur jeu de donnees reel avec diagnostic d'autocorrelation.

## Outils
Python — NumPy, pandas, scipy, matplotlib
